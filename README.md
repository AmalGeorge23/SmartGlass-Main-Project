Visually impaired individuals face significant challenges in independently navigating their environment, identifying objects and reading printed text which often leads to increased dependence on others and reduced personal safety. To address these limitations, this project proposes a Smart Glass system that provides real-time assistive support through an intelligent, wearable platform. The system integrates face recognition, object identification, obstacle detection with audio alerts, and reading assistance. A Raspberry Pi Zero 2 W mounted on the glasses, along with a Pi Camera Module 3, captures visual data, which is transmitted via a secure tunneling service to a remote processing server for computationally intensive tasks, enabling efficient performance despite on-device resource constraints. The processed results are returned to the wearable unit and delivered to the user through auditory feedback, while obstacle detection and immediate alerts are handled locally to ensure low-latency response. The system further incorporates voice command-based mode selection, health monitoring using a MAX30102 sensor for heart rate and blood oxygen saturation (SpO2), and location tracking using a GP-02 GNSS kit. Health data and real-time location are logged to the cloud and accessed through a mobile application, allowing guardians to monitor the user’s status and receive automated SOS alerts during abnormal health conditions. The smart glass frame is ergonomically designed and fabricated using 3D printing to enhance comfort and usability. The proposed solution aims to deliver a reliable, user-friendly, and scalable assistive technology that enhances independence, safety, and overall well-being for visually impaired individuals.
